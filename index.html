Nomoogle

Chrome Extension
Firefox Extension

Installation

  • Clone or Download the repo.
  • Go to Extension settings
  • Enable Developers Mode
  • Click on "Load Unpackaged" and add the folder you downloaded.

Features

Get google alternatives in one click
Get popup Reminder, Everytime you use a google product
With two special modes
Strict Mode:

Block the entire webpage completly, doesn't allow you to move forward.
